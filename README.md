## 🧪 The Protocol
*Project: Glass House* is an experimental psychological horror simulation designed for the 30-day isolation challenge. As the monitor, you must navigate the Subject (Vesper) through a decaying digital environment. 

### ⚠️ Warning
As the days progress, system stability will decrease. Do not attempt to force-close the interface during a **DATA_PURGE**.

## 🛠️ Technical Features
- **Procedural Audio Synthesis:** Real-time "Mental Static" generated via the Web Audio API (no external MP3s required).
- **Adaptive Glitch Engine:** Frame-buffer manipulation and canvas tearing that scales with the in-game "Stability" metric.
- **Hybrid Input System:** Custom touch-gesture logic for mobile and standard WASD/Space for desktop.
- **Persistence Layer:** Integrated `localStorage` to track subject degradation across sessions.
